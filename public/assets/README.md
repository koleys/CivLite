# CivLite Assets

This folder contains game assets. Replace placeholder files with real assets.

## Required Assets

### Images
Place in `public/images/`:
- `terrain/` - Terrain tile images (64x64 recommended)
- `units/` - Unit icons (64x64 recommended)  
- `icons/` - UI icons (32x32 recommended)

### Audio
Place in `public/audio/`:
- `music/` - Background music (MP3/OGG)
- `sfx/` - Sound effects (MP3/OGG)

## Recommended Sources
- Kenney Assets (CC0): https://kenney.nl/assets
- OpenGameArt: https://opengameart.org
- Game-Icons.net: https://game-icons.net

## Fallback Audio
The game includes programmatic fallback sounds if audio files are not provided.
